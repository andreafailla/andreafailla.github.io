Pixel Sonification Lab

Pix 2 Beats

Pix2Beats samples pixel colors, maps hues to notes in a chosen scale, and builds a stereo waveform that is shaped with pedalboard-style effects - all locally in your browser.
This is a browser-based demo that approximates the functionality of the original Pix2Beats project I built with Streamlit. Originally, Pix2Beats was a Python application hosted on Streamlit Cloud, and it used the Spotify's Pedalboard library for audio processing. This web version is a reimagining of the concept, built with JavaScript and Web Audio API, and it runs entirely in the browser without any server-side processing. The effects are approximations of the original pedalboard chain, adapted for in-browser playback.
